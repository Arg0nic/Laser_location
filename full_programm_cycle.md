Ниже — полный цикл программы “от начала до конца” в той схеме, которую вы выбрали: **двухступенчатый критерий** + **Monte-Carlo по wander** +  **шум + EWMA** .

---

## 0) Инициализация (задаём константы и параметры)

* Размер цели: (d_{\text{target}}=0.5\ \text{м}), радиус цели (R_t=0.25\ \text{м})
* Базовая расходимость: (\theta_0=0.1\ \text{мрад}=10^{-4}\ \text{рад})
* Закон турбулентности: (\theta(L)=\theta_0\cdot 3^{\lfloor L/200\rfloor})
* Диапазон расстояний: (L\in[L_{\min},L_{\max}]) с шагом (\Delta L)
* Monte-Carlo: число прогонов (N) (например 10000)
* Wander: (\sigma_w(L)) (или фиксированное (\sigma_w) для сценария)
* Порог энергии на цели: (\eta_{\min}=0.8)
* Шум по амплитуде: фон (b), СКО шума (\sigma_A)
* Детект-порог: (T)
* EWMA: коэффициент (\alpha)
* Число импульсов в одном прогоне для фильтрации: (M) (например 10–50)

---

## 1) Внешний цикл по расстоянию (L)

Для каждого расстояния (L):

### 1.1) Считаем расходимость и размер пятна

1. (k=\lfloor L/200\rfloor)
2. (\theta(L)=\theta_0\cdot 3^k)
3. Диаметр пятна:
   [
   d(L)=\theta(L),L
   ]
   (если добавляете начальный диаметр (d_0): (d(L)=\sqrt{d_0^2+(\theta(L)L)^2}))
4. Радиус пятна: (R_s(L)=d(L)/2)

### 1.2) 1-й этап (геометрический фильтр из условия)

Если (d(L)>0.5\ \text{м}):

* сразу фиксируем (p(L)=0) (в этом режиме по условию “гарантия” отсутствует)
* переходим к следующему (L)

Если (d(L)\le 0.5\ \text{м}):

* идём в Monte-Carlo.

---

## 2) Внутренний цикл Monte-Carlo для фиксированного (L)

Заводим `success = 0`.

Повторяем (i=1..N):

### 2.1) Генерируем дрожание центра (beam wander)

1. Генерируем
   [
   X_i, Y_i \sim \mathcal{N}(0,\sigma_w(L)^2)
   ]
2. Смещение центра:
   [
   r_i=\sqrt{X_i^2+Y_i^2}
   ]

### 2.2) Считаем долю энергии пятна, попавшую в цель

1. Вычисляем площадь пересечения кругов “пятно–цель” (S_{\cap}(R_s,R_t,r_i))
2. Доля энергии (модель равномерного пятна):
   [
   \eta_i=\frac{S_{\cap}}{\pi R_s^2}
   ]

### 2.3) 2-й этап часть 1 (порог по энергии)

Если (\eta_i < \eta_{\min}):

* сразу “провал” в этом прогоне → переходим к следующему (i)

Если (\eta_i \ge \eta_{\min}):

* моделируем приём и детект.

### 2.4) Моделируем шумный сигнал и применяем EWMA

Мы считаем, что в одном прогоне дальномер посылает серию из (M) импульсов.

Для (t=1..M):

1. “Идеальная” амплитуда:
   [
   A_{t,\text{ideal}} = A_0\cdot \eta_i
   ]
2. Шумное измерение:
   [
   A_t = A_{t,\text{ideal}} + b + \varepsilon_t,\quad \varepsilon_t\sim \mathcal{N}(0,\sigma_A^2)
   ]
3. Экспоненциальное сглаживание (EWMA):
   [
   \hat A_t=\alpha A_t+(1-\alpha)\hat A_{t-1}
   ]
   (первое значение можно взять (\hat A_1=A_1))

После (M) импульсов получаем (\hat A_M).

### 2.5) 2-й этап часть 2 (порог чувствительности)

Если (\hat A_M \ge T):

* считаем прогон успешным → `success += 1`
  иначе — провал.

---

## 3) Вероятность успеха на расстоянии (L)

После (N) прогонов:
[
p(L)=\frac{\text{success}}{N}
]
Сохраняем (p(L)) для этого (L).

---

## 4) Итоговый результат

После прохода всех расстояний:

* строим зависимость (p(L))
* выбираем рабочую дальность по требованию надёжности, например:
  [
  L_{\text{work}}=\max{L:\ p(L)\ge 0.95}
  ]
